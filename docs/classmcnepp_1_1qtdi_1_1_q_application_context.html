template<typename S , LookupKind lookup = LookupKind::STATIC>
ProxyRegistration< S > mcnepp::qtdi::QApplicationContext::getRegistration ( ) const

Obtains a ProxyRegistration for a service-type.
In contrast to the ServiceRegistration that is returned by registerService(), the ServiceRegistration returned by this function is actually a Proxy.
This Proxy manages all Services of the requested type, regardless of whether they have been registered prior to invoking getRegistration().
This means that if you subscribe to it using ServiceRegistration::subscribe(), you will be notified about all published services that match the Service-type.

LookupKindType of managed Services
LookupKind::STATICThose Services whose Registration::service_type() matches the requested typeid(S)
LookupKind::DYNAMICThose Services that can be converted to the requested type using dynamic_cast<S*>().
Template Parameters
Sthe required service-type.
lookupdetermines the strategy for retrieving the ServiceRegistration-Proxy. See the table above for details!
Returns
a ProxyRegistration that corresponds to all registration that match the service-type.

◆ publish()

virtual bool mcnepp::qtdi::QApplicationContext::publish ( bool  allowPartial = false)
pure virtual

Publishes this ApplicationContext. This method may be invoked multiple times. Each time it is invoked, it will attempt to instantiate all yet-unpublished services that have been registered with this ApplicationContext.

Parameters
allowPartialhas the default-value false, this function will return immediately when a service cannot be published (due to missing dependencies, unresolveable properties, etc...). Additionally, the cause of such a failure will be logged with the level QtMsgType::QtCriticalMessage. if allowPartial == true, the function will attempt to publish as many pending services as possible. Failures that may be fixed by further registrations will be logged with the level QtMsgType::QtWarningMessage.
Returns
true if there are no fatal errors and all services were published (in case allowPartial == false), or at least one service was published (in case allowPartial == true).

Implemented in mcnepp::qtdi::StandardApplicationContext.
