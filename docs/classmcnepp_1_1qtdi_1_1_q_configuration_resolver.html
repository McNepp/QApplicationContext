virtual QVariant mcnepp::qtdi::QConfigurationResolver::getConfigurationValue ( const QString & key,
bool searchParentSections = false ) const
nodiscardpure virtual

Retrieves a value from the ApplicationContext's configuration.
This function will be used to resolve placeholders in Service-configurations. Whenever a placeholder shall be looked up, the ApplicationContext will search the following sources, until it can resolve the placeholder:

  1. The environment, for a variable corresponding to the placeholder.
  2. The instances of QSettings that have been registered in the ApplicationContext.
    See also
    mcnepp::qtdi::config()
    Parameters
    keythe key to look up. In analogy to QSettings, the key may contain forward slashes to denote keys within sub-sections.
    searchParentSectionsdetermines whether the key shall be searched recursively in the parent-sections.
    Returns
    the value, if it could be resolved. Otherwise, an invalid QVariant.
