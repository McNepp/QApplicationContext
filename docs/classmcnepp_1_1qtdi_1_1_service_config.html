Detailed Description

template<typename S>
class mcnepp::qtdi::ServiceConfig< S >

A type-safe service-configuration.
This class is the type-safe equivalent to the service_config.
In contrast to service_config, configuration-values are specified using method-pointers, not property-names.
Also, there does not have to be a Q_PROPERTY declared for each configuration-value. You may use the syntax for private properties, i.e. a name preceded by a dot to indicate that.
