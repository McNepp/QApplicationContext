Detailed Description

template<typename C>
class mcnepp::qtdi::ApplicationContextImplBase< C >

Extensible Implementation of QApplicationContext.
This class provides a means of implementating the interface QApplicationContext with the potential for additional functionality.
The "canonical way" of instantiating QApplicationContext is by means of the class StandardApplicationContext.
However, should you want to provide a customized implementation, or should you want to implement additional interfaces, ApplicationContextImplBase may be put to use.

Template Parameters
