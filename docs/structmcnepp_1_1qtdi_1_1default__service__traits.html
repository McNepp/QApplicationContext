Detailed Description

template<typename S, typename I = std::nullptr_t, ServiceInitializationPolicy serviceInitPolicy = ServiceInitializationPolicy::DEFAULT, typename F = service_factory<S>>
struct mcnepp::qtdi::default_service_traits< S, I, serviceInitPolicy, F >

Provides default-values for service_traits.
Specializations of service_traits in client-code are encouraged to extend this type!
this template provides the following declarations:

  • service_type an alias for S.
  • initializer_type an alias for std::nullptr_t.
  • static constexpr ServiceInitializationPolicy initialization_policy = ServiceInitializationPolicy::DEFAULT;
  • factory_type an alias for service_factory.
Template Parameters
Sthe service-type
Ithe type of the initializer. Defaults to std::nullptr_t.
serviceInitPolicydetermines whether the initializer will be invoked before publication (the default) or after it.
Fthe type of the service-factory. Defaults to mcnepp::qtdi::service_factory.
