Detailed Description

template<typename... S>
class mcnepp::qtdi::ServiceCombination< S >

A combination of Services that can be subscribed.
An instance of this type will be returned by mcnepp::qtdi::combine().
This type is neither copyable nor moveable.
It is meant to be treated solely as a temporary object, on which ServiceCombination::subscribe() shall be invoked.
