Detailed Description

template<typename S>
class mcnepp::qtdi::ServiceRegistration< S >

A type-safe wrapper for a detail::ServiceRegistration. Instances of this class are being produces by the public function-templates QApplicationContext::registerService() and QApplicationContext::registerObject().
