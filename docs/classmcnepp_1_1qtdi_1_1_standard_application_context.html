Parameters

If allowPartial == true, the function will attempt to publish as many pending services as possible. Failures that may be fixed by further registrations will be logged with the level QtMsgType::QtWarningMessage.
Returns
true if there are no fatal errors and all services were published.

Thread-safety: This function may only be called from the ApplicationContext's thread.

Parameters

objNamethe name for this Object in the ApplicationContext. Note: this name will not be set as the QObject::objectName(). It will be the internal name within the ApplicationContext only.
Template Parameters
Sthe primary service-type for the object.
IFacesadditional service-interfaces to be advertised. If a type appears more than once in the set of types comprising S and IFaces, compilation will fail with a diagnostic.
Returns
a ServiceRegistration for the registered service, or an invalid ServiceRegistration if it could not be registered.
